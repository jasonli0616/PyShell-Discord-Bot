# PyShell-Discord-Bot

## What does this do?
Add this bot to your Discord server, for a python shell right in a channel.

## How do I use this?
Bot no longer available due to people removing py files with subprocess, only available through source code
1. Copy/fork source code
2. Make Discord bot, add to server
3. Create channel named "#pyshell" or "#bot-code"
4. Type python code into channel!
